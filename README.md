# Calculadora de Nota Fiscal de Serviço (NFS-e)

Este projeto é um exercício prático que simula o cálculo e a visualização de uma Nota Fiscal de Serviço (NFS-e) baseada em valores e alíquotas de impostos fornecidos pelo usuário.

## Demonstração e Uso

A aplicação permite a entrada dos seguintes dados para o cálculo:

* **Valor da Venda**
* **Descrição dos Serviços**
* **Alíquotas de Impostos (IRPF, PIS, COFINS, INSS, ISSQN)**

Após o cálculo, a página exibe o **Valor Líquido a Receber** e o detalhamento de todos os impostos.

## Tecnologias Utilizadas

* **HTML5:** Estrutura da página e formulário.
* **CSS3:** Estilização responsiva e aplicação da paleta de cores.
* **JavaScript:** Lógica de cálculo dos impostos, validação de dados e manipulação do DOM para exibir a NFS-e.

## Instalação (Local)

Para rodar este projeto em sua máquina local, siga os passos:

1.  **Clone o repositório:**
    ```bash
    git clone https://github.com/Eduardolimzz/calculadora-nfse.git
    ```
2.  **Acesse o diretório do projeto:**
    ```bash
    cd calculadora-nfse
    ```
3.  Abra o arquivo `index.html` (ou o nome que você usou para o arquivo HTML) em seu navegador preferido.
